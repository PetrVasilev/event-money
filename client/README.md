
# Клиентская часть рроекта для хакатона ВКонтакте Якутск

### Установка проекта

Чтобы запустить проект:
- Скачайте проект 
```sh
$ git clone https://github.com/PetrVasilev/event-money.git
```
- Перейдите в каталок клиентской части
```sh
$ cd client
```
- Установите все npm пакеты
```sh
$ npm install
```
или 

```sh
$ yarn install
```
- Запустите проект
```sh
$ npm start
```
или 

```sh
$ yarn start
```
